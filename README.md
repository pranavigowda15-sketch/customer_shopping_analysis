# Customer Shopping Behavior Analysis

## Overview
The Customer Shopping Behavior Analysis project focuses on understanding customer purchasing patterns using retail transaction data. The project analyzes how factors such as subscriptions, discounts, demographics, and purchase frequency influence customer spending and revenue generation. The insights help businesses improve customer engagement and optimize marketing strategies.

## Features
- End-to-end data analytics workflow
- Customer segmentation based on demographics and behavior
- Revenue analysis by gender and category
- Subscriber vs non-subscriber comparison
- Discount impact and repeat buyer analysis
- SQL-based analytical queries
- Interactive dashboards using Power BI

## Technologies Used
- Python
- Pandas
- NumPy
- PostgreSQL
- SQLAlchemy
- psycopg2
- SQL
- Power BI
- Git and GitHub

## How It Works
1. Raw customer shopping data is loaded and cleaned using Python.
2. Data preprocessing is performed, including handling missing values and feature engineering.
3. The cleaned dataset is stored in a PostgreSQL database.
4. SQL queries are executed to analyze customer behavior and revenue patterns.
5. Results are visualized using interactive Power BI dashboards.

## Key Analyses
- Revenue generated by gender
- Spending comparison between subscribers and non-subscribers
- Discount usage impact on purchases
- Repeat buyer identification
- Top products within each category
- Purchase frequency analysis
